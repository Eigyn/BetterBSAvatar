# Security Policy

## Supported Versions

BetterBSAvatar is experimental and currently targets Beat Saber 1.40.8 only.

## Reporting Issues

Please report security-sensitive issues privately when possible. Do not include private logs, access tokens, local account paths, or copied game files in public reports.

## Source Safety Notes

This repository should contain source code only. Beat Saber binaries, decompiled IL, copied dependencies, generated build outputs, and local configuration files should stay out of git.
